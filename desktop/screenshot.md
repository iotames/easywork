## 简介

介绍场景的屏幕截图工具，用于快速截图，标注，贴图等。

## 跨平台

| 特性对比 | Snipaste | PixPin |
|---------|----------|--------|
| **官方地址** | [https://zh.snipaste.com/](https://zh.snipaste.com/) | [https://pixpin.cn/](https://pixpin.cn/) |
| **跨平台支持** | Windows、macOS、Linux | Windows、macOS |
| **核心功能** | 截图、贴图、标注 | 截图、贴图、长截图、GIF录屏、OCR文字识别 |
| **贴图特性** | 支持将图片、文字、颜色信息等转为贴图；支持缩放、旋转、设置透明度、鼠标穿透 | 支持将图片、文字、文件、颜色代码等作为贴图 |
| **标注工具** | 矩形、椭圆、箭头、铅笔、马赛克、高斯模糊等 | 矩形、线条、箭头、文字等，支持序列号标注 |
| **OCR功能** | 不支持 | 支持，可快速识别并复制图片中的文字 |
| **长截图** | 不支持 | 支持，通过智能拼接算法实现 |
| **录屏功能** | 不支持 | 支持，可录制屏幕并保存为GIF动态图 |
| **授权模式** | 个人用户免费，企业需购买专业版 | 免费软件，提供会员服务 |


## Windows

- 系统自带的快捷键：`Win` + `Shift` + `S`
