项目简介

EasyWork 工作和学习的备忘笔记，开发和运维的效率工具。

目录结构

  • easywork
    • backup: 数据备份
      • rsync.sh: 从远程服务器进行增量备份
    • config: 配置相关
      • vimrc.conf: .vimrc配置文件
    • clang: C语言相关
      • README.md: C语言开发必读
      • makefile-cmake-xmake.md: Makefile,Cmake和xmake
      • gcc.md: GCC编译器
    • desktop: 桌面工具
      • windows_terminal: Windows Terminal相关
      • browser_run.sh: 从命令行运行浏览器
      • browser_use_proxy.bat: 连接某个HTTP代理后打开浏览器
      • set_dns.bat: DNS一键设置工具
    • devops: DevOps通用开发和运维
      • gitlab: Gitlab主题
      • cicd.sh: 持续集成脚本
      • gitlab-ci.yml: Gitlab CI/CD流水线配置
      • redis.md: Redis相关
      • setenv.md: 设置环境变量
    • devstd: 开发标准和架构
      • datastruct.md: SQL和数据
      • erpsysarch.md: 系统架构设计
    • docker: 容器技术
      • compose: Docker Compose相关
      • harbor: Harbor私有镜像仓库
      • mysql: MySQL相关
      • docker_run.sh: Docker运行脚本实例
      • volumes.md: Docker卷的使用
    • etl: ETL主题
    • git: Git代码管理工具
      • conf.sh: 常用Git配置
      • gitflow.md: Gitflow标准工作流
    • golang: Go语言相关
    • java: Java相关
    • linux: Linux系统相关
      • disk.md: 磁盘分区和挂载
    • net: 网络基础知识和攻防实践
      • httpheader.md: HTTP请求头和网络攻防
      • netlayer.md: 互联网分层协议和数据封装
    • nginx: Nginx主题
    • nodejs: NodeJS相关
      • docs.md: 文档工具
    • other: 其他
    • sql: SQL主题
      • create_db_user.sql: 创建数据库和用户
    • ssl: 证书和自动续期
    • weixin: 微信相关
      • wxwork: 企业微信相关代码脚本
