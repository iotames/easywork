## Agent Skills 简介

**Agent Skills** 是一种标准化的、可移植的 AI 能力扩展包，将专业知识、工作流程和脚本封装成文件夹形式（包含 `SKILL.md` 等文件），供 AI Agent 按需加载和执行。

[agentskills.io](https://agentskills.io/) 是 Anthropic 于 2025年12月18日 发起的开放标准，规范了 Skill 的目录结构、元数据格式和执行约定，旨在实现技能在不同 AI Agent（如 Claude Code、Cursor、OpenClaw 等）之间的互操作与复用。
