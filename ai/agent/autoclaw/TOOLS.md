# TOOLS.md —— 本地笔记

技能（Skill）定义工具_怎么用_。这个文件是给_你_自己的——你的环境特有的东西。

## 这里放什么

比如：

- 摄像头名称和位置
- SSH 主机和别名
- 偏好的 TTS 语音
- 音箱/房间名称
- 设备昵称
- 任何与环境相关的东西

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，移动触发

### SSH

- home-server → 192.168.1.100，用户：admin

### TTS

- 首选语音：Nova（温暖，略带英音）
- 默认音箱：厨房 HomePod
```

## 为什么分开？

技能是共享的。你的配置是你自己的。把它们分开意味着你可以在不丢失笔记的情况下更新技能，也可以在不暴露你的基础设施的情况下分享技能。
