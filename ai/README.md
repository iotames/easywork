## HuggingFace开源平台

Hugging Face 是一个 `开源AI社区和平台`，专注于自然语言处理（NLP）和机器学习。提供一系列核心工具和服务，极大地降低了 AI 模型开发、共享和部署的门槛，是当前 AI 应用生态中不可或缺的基础设施。

- 学习：https://huggingface.co/learn
- python的huggingface_hub客户端：https://huggingface.co/docs/huggingface_hub/main/cn/quick-start

核心服务：

1. Transformers 库：一个基于 PyTorch、TensorFlow 和 JAX 的流行开源库，提供了数千个预训练模型（如 BERT、GPT、T5）的简易调用接口，是 NLP 研究和应用的事实标准。
2. Model Hub（模型中心）：一个庞大的模型仓库，社区成员可以上传、下载、共享和发现各种 AI 模型，涵盖了文本、图像、音频等多模态任务。
3. Datasets 库与数据集中心：提供高效的数据集加载工具和一个共享数据集的平台，方便研究人员和开发者获取高质量的训练数据。
4. Spaces：一个免费的机器学习应用托管平台，允许用户快速构建、分享和部署交互式 AI 演示应用（如 Gradio 或 Streamlit 应用）。
5. Inference Endpoints：企业级服务，用于大规模部署和管理模型推理 API。
